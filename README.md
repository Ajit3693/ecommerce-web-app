# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application built using React.js, Spring Boot, and MySQL. The platform enables users to browse products, manage carts, place orders, and securely manage their accounts.

## 🚀 Features

- User Registration & Login
- Product Listing & Search
- Product Categories & Filtering
- Shopping Cart Management
- Order Placement & Tracking
- Admin Dashboard
- REST API Integration
- Responsive Design
- Secure Authentication
- MySQL Database Management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ecommerce-web-app.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```
